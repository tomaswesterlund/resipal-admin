# Resipal Admin

Resipal Admin is the "command center" for neighborhood and building managers. It’s a Flutter-built dashboard designed to take the headache out of running a community.

Instead of chasing down receipts or digging through spreadsheets, we’ve put everything in one place. You can quickly check who’s paid their maintenance fees, approve new neighbor requests, and keep an eye on which properties are sitting vacant or have a mounting debt.
